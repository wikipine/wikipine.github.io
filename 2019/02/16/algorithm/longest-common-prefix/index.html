最长公共前缀

算法 leetcode 2019/02/16

题目来源：https://leetcode-cn.com/problems/longest-common-prefix/

题目
编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 “”。

示例

输入: [“flower”,”flow”,”flight”]
输出: “fl”

输入: [“dog”,”racecar”,”car”]
输出: “”
解释: 输入不存在公共前缀。

分析
取出数组的第一个值，然后循环字符串，循环数组，单个判断，一旦不符合，退出循环，返回数据
